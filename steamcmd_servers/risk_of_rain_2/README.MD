# Risk of Rain 2
Risk of Rain 2 is a Rougelite, thirdperson action game. It features tons of items and multiple characters. Playable in both singleplayer and multiplayer.

### Server Ports
Risk of Rain 2 requires up to 2 ports.

| Port    | default       |
|---------|---------------|
| Server  |  27015  	  |
| Query   |  27016        |
