# Risk of Rain 2
Rising World is a voxel based open-world sandbox game, featuring a procedurally generated world, playable in single and multi-player.

### Server Ports
Risk of Rain 2 requires up to 3 ports.

| Port    | default       |
|---------|---------------|
| Server  |  27015  	  |
| Query   |  27016        |
| Steam   |  0            |
